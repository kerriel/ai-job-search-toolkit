# Applications Tracker

Track your job applications in one place. Claude will help maintain this file if your project instructions are set up. You can also update it manually.

Update the status column as you progress through each application. Add new rows as you apply.

### Status values

- **Applied** - application submitted
- **Phone screen** - initial call scheduled or completed
- **Interview** - progressed to interview stage
- **Offer** - offer received
- **Rejected** - application unsuccessful
- **Withdrawn** - you withdrew from the process

### Posted By

Use "Direct" for roles posted by the employer. For recruiter-sourced roles, note the recruiter name and agency (e.g. "Jane Smith, Hays").
